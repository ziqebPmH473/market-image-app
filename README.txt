Market Image Generator

- Serverless (client-side only)
- Paste TSV blocks (NotebookLM output) and export 1080x1080 PNG

How to use:
1) Open index.html in a browser (or host as static site)
2) Paste TSV into the textarea
3) Click PNG保存

Notes:
- For crisp output, export is generated at 2160x2160.
- If the font looks different across devices, that's normal in serverless mode.
